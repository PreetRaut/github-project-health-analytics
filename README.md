# SaaS Revenue Analytics & AI Insights Platform

This project simulates a modern analytics engineering workflow for a B2B SaaS company.

## What it does

- Generates synthetic customer and subscription data
- Stores data in DuckDB
- Uses dbt to transform raw data into analytics-ready models
- Calculates MRR, ARR, churn rate, and customer metrics
- Builds a Streamlit dashboard for revenue analytics
- Demonstrates semantic-layer thinking and data quality testing

## Tools Used

- Python
- SQL
- dbt
- DuckDB
- Streamlit
- Plotly
- GitHub

## Key Metrics

- Monthly Recurring Revenue
- Annual Recurring Revenue
- Churn Rate
- Customer Segmentation
- Regional Revenue Performance

## Why this project matters

This project demonstrates analytics engineering, business intelligence, revenue analytics, and data modelling skills relevant to SaaS analytics roles.
